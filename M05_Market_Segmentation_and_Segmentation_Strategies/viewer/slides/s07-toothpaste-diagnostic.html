All four segments share the same core need (prevent decay). What drives differentiation? What does this imply for a firm competing in only one segment?

Key Insight

Needs-based segments require different products, prices, and messages even when the core problem is identical. The differentiating benefit — not the core need — defines what each segment actually buys and which program serves it.
