Learning Objectives

LO1–LO6
LO1 Analysis Classify a communication as brand-image, brand-information, or brand-action by analyzing its objective, desired response, and performance metric.
LO2 Application Compute the Customer Response Index (CRI) from a five-stage hierarchy and identify which stage improvement yields the highest marginal CRI gain.
LO3 Application Calculate GRPs from reach and frequency; compute weighted average message frequency; evaluate media selection tradeoffs using CPM.
LO4 Analysis Distinguish pull from push communications; evaluate conditions under which each strategy creates or destroys supplier profit.
LO5 Application Compute advertising elasticity; project current-period and total multi-period sales impact using the advertising carryover formula.
LO6 Evaluation Assess the profit impact of a proposed communications investment by integrating CRI, GRP, carryover, and push/pull tradeoff data into a defensible recommendation.
