MKT 472 · Module 6 · App 1

Competitive Advantage Profiler

Threshold-based analysis of cost, differentiation, and market strength.

Learning Goal

Evaluate whether cost, differentiation, and market position each cross a true advantage threshold. Compare index statuses to distinguish genuine advantage from parity. Use sensitivity scenarios to test how metric changes alter profit implications.

Worksheet Quiz Instructions

  1. Read the Canvas quiz question first.
  2. Follow the preset, scenario, or modifications specified in the question.
  3. Use this app to perform the required calculations and analysis.
  4. Enter your answers in the Canvas quiz.
  5. Answers are submitted through Canvas, not through this app.

Required for Quiz

The Canvas quiz question determines which preset, scenario, or modifications to use. If a question does not specify otherwise, begin with the default preset shown in this app. Default preset: Vantage Industrial.

How to Use This App
  1. Choose a preset or enter input values for cost, differentiation, and market share fields.
  2. Select direct DI entry or the 3-benefit DI calculation mode.
  3. Review output cards for VCAI, DI, RMS, and corresponding status labels.
  4. Use the sensitivity panel to compare before/after metric and profit implication changes.
