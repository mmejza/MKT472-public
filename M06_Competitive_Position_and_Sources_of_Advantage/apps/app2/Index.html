Industry Attractiveness Scorer

Scores represent attractiveness contribution after direction correction.

Learning Goal

Evaluate industry attractiveness by scoring each force using the correct direction logic. Compare how force-level changes alter overall IAI classification and recommendation. Use side-by-side comparison to identify relative attractiveness across industries.

Worksheet Quiz Instructions

  1. Read the Canvas quiz question first.
  2. Follow the preset, scenario, or modifications specified in the question.
  3. Use this app to perform the required calculations and analysis.
  4. Enter your answers in the Canvas quiz.
  5. Answers are submitted through Canvas, not through this app.

Required for Quiz

The Canvas quiz question determines which preset, scenario, or modifications to use. If a question does not specify otherwise, begin with the default preset shown in this app.

How to Use This App
  1. Select a preset and adjust force sliders for Industry A, and optionally enable Industry B.
  2. Set each force score so positive values represent more attractive conditions.
  3. Review IAI, classification, recommendation, and running-sum output lines.
  4. Use radar charts and comparison output to evaluate relative attractiveness.
