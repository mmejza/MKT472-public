Scale Effect vs. Scope Effect
Honda Ignition Switch — same component, two different cost mechanisms
Scale Effect
Cost per unit falls as volume increases
· Honda doubles auto production capacity C1 → C2
· Unit cost curve shifts DOWN as capacity rises
· Same product, more of it → cheaper per unit
Do more of the same thing.
Scope Effect
Cost per unit falls as product breadth increases
· Honda uses same ignition switch across 5 product lines
· Autos · Motorcycles · Lawnmowers · Snowblowers · Pumps
· Buys more switches in total → lower cost per switch
Use the same thing for more products.
Takeaway

Scale = do more of the same thing.  Scope = use the same thing for more products. Both lower cost per unit — but through completely different mechanisms.
