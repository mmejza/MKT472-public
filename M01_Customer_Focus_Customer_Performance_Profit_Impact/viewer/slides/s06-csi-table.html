How to Calculate Weighted-Average CSI

Multiply each tier's CSI value by its weight (as a decimal) and sum across all tiers.
Example: 0.25×100 + 0.35×80 + 0.20×60 + 0.15×40 + 0.03×20 + 0.02×0 = 72

Gate 2 Note

Gate 2 will give you a different distribution. The method stays the same — practice calculating the weighted average from these tiers before the exam.
