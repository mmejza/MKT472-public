Brand Growth Strategies Matrix
Four strategies — defined by two decisions: brand name and product class
Existing Product Class
New Product Class
Existing Brand
Line Extension
Additional items added to an existing product line under the current brand name — same class, same brand, new variety
Brand Extension
Current brand name used in a different product class — same brand, new market. Leverages equity across categories.
New Brand
Fighter / Flanker Brand
New defensive brand in an existing class — created to protect the main brand from competitive attack at a different price point
Gate 2 Trap
New Brand
Entirely new brand in a product class not previously served — no equity leverage; maximum investment required
Gate 2 Alert — Fighter/Flanker vs. Brand Extension

Fighter/Flanker = new brand, existing class. Brand Extension = existing brand, new class. Students confuse these. Know the distinction cold.
