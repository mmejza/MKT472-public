MKT 472 · Module 7 · App 2

Brand Equity Scorecard Calculator

Build intuition for brand assets, liabilities, equity, and temporary liability spikes.

Learning Goal

Evaluate how brand assets and liabilities combine into an overall equity score. Interpret how temporary liability spikes can rapidly reduce equity even when assets are strong. Use tab outputs to compare baseline and post-spike equity conditions.

Worksheet Quiz Instructions

  1. Read the Canvas quiz question first.
  2. Follow the preset, scenario, or modifications specified in the question.
  3. Use this app to perform the required calculations and analysis.
  4. Enter your answers in the Canvas quiz.
  5. Answers are submitted through Canvas, not through this app.

Required for Quiz

The Canvas quiz question determines which preset, scenario, or modifications to use. If a question does not specify otherwise, begin with the default preset shown in this app.

How to Use This App
  1. Set asset and liability sliders using the allowed 0, 25, 50, 75, and 100 values.
  2. Review tab totals and horizontal bars for assets and liabilities.
  3. Open Brand Equity to compare assets, liabilities, and computed equity.
  4. Use Spike Simulator to test temporary liability shocks without changing original sliders.
