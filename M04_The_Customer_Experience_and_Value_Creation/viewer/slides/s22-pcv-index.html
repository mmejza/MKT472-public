PCV Index: Assembly and Result
PCV Index = Overall Performance Index − Cost of Purchase Index
Component Index Value Interpretation
Product Performance Index 137 37% above competitive average — strong
Service Quality Index 90 10% below competitive average — weakness
Brand Reputation Index 140 40% above competitive average — strong
Overall Performance Index 123 (137×0.60) + (90×0.30) + (140×0.10) = 82.2 + 27.0 + 14.0 = 123
Cost of Purchase Index 103 3% more expensive than competitive average
PCV Index 20 123 − 103 = 20 → Superior Value Zone
Superior Value Zone

PCV Index of +20 = superior value zone. Ch. 4 Fig 4-26: companies with positive PCV indexes produce above-average profitability. Negative PCV = inferior value zone = below-average profitability. The service quality weakness (90) is partially offset by strong brand reputation (140) — but remains the firm's priority improvement area.
