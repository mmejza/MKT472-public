MKT 472 Module 4

Conjoint Analysis CVI Simulator

Use conjoint preference values to compare customer value versus profitability across two strategies.

Focus mode off

Learning Goal

Identify how strategy design affects CVI and test whether the highest customer-value option is also the highest profitability option. Use this to spot the value-margin paradox in conjoint decisions.

How to Use This App
  1. Confirm the Silent Floor preset, then review each attribute’s preference levels.
  2. Select levels for Strategy A and Strategy B and set each strategy price premium.
  3. Read CVI values, value drivers, and the comparison flag in the dashboard.
  4. Use preference and CVI charts to visualize why one strategy ranks higher.

Worksheet Quiz Instructions

  1. Use this app to perform the required calculations.
  2. Follow the required preset or scenario.
  3. Record your final answers in the Canvas quiz questions below.
  4. Answers are submitted through Canvas, not through this app.

Required for Quiz

Load the Silent Floor preset and begin with the default scenario before making modifications.
