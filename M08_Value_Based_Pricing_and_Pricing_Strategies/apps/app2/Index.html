MKT 472 · Module 8 · App 2

Performance Value Pricing Simulator

Compute weighted value scores by segment and test pricing strategy outcomes.

Learning Goal

Evaluate how segment-specific weighting changes firm-versus-competitor value outcomes. Compare value advantage across pricing strategies using VP scenarios and adjusted value scores. Interpret when the same product profile yields opposite strategic conclusions under different segment priorities.

Worksheet Quiz Instructions

  1. Read the Canvas quiz question first.
  2. Follow the preset, scenario, or modifications specified in the question.
  3. Use this app to perform the required calculations and analysis.
  4. Enter your answers in the Canvas quiz.
  5. Answers are submitted through Canvas, not through this app.

Required for Quiz

The Canvas quiz question determines which preset, scenario, or modifications to use. If a question does not specify otherwise, begin with the default preset shown in this app. Default preset: Quality-Conscious.

How to Use This App
  1. Load a segment preset or set custom weights that always sum to 100%.
  2. Enter firm and competitor preference scores plus price inputs.
  3. Review value score summary and VP1-VP5 table changes across price points.
  4. Use the value map and verdict output to compare strategic fit by segment.
