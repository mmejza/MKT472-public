Marketing ROI: The 100% Floor
ROI = 100% is break-even — the program recovers its cost but generates no profit

The Floor Rule
ROI = 100% → NMC = MSE (break-even)
ROI < 100% → NMC < MSE → program is net-negative
ROI > 100% → NMC > MSE → program is profitable

The floor is not zero — it is 100%. A 50% ROI means the firm is spending $2 in marketing to recover only $1 in net contribution.
Gate 2 Q3 — Floor Test
Calculate: ROI = NMC ÷ MSE × 100. Then classify: is the result above or below 100%?

If ROI < 100% → the marketing program is not recovering its own cost. This is distinct from negative NMC — it is possible for NMC to be positive yet ROI to be below 100% if MSE is very large relative to NMC.
